# Disaster Relief Project

A comprehensive web application designed to manage disaster relief efforts, including victim registration, resource allocation, and team management.

## Features
- **User Panel**: For victims and survivors to report incidents and request aid.
- **Admin Panel**: For coordinators to manage resources, teams, and relief centers.
- **Real-time Tracking**: Monitor disaster incidents and relief progress.
- **Resource Management**: Efficiently allocate supplies and personnel.

## Tech Stack
- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
- **Backend**: Java / MySQL
- **Database**: MySQL

## Setup Instructions
1. Import the `schema.sql` file into your MySQL database.
2. Update the database connection strings in the source code if necessary.
3. Run the application using `run.bat`.

## Database Schema
The system uses a relational database with the following key entities:
- Incidents
- Victims
- Relief Requests
- Resources
- Teams
- Centers
